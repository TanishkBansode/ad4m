# AD4M Launcher

A desktop application to handle the running and administration of an AD4M user.

## Development

Run frontend dev server,

```shell
pnpm install
pnpm start
```

In another terminal, run the tauri app,

```shell
pnpm tauri dev
```

## Building

```
pnpm install
pnpm run build-macos/windows/linux (choose your correct OS)
```
